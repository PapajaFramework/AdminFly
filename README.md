# AdminFly 
## Spring MVC Admin Panel

### Short Description

* Support multilanguage
    - English (by default)
    - Українська
    - Русский
    - Portuguese
    - Spanish
    - German
    - French
    - Polish
    - Norwegian
    - Indonesian
    - বাংলা (Bengali)
    - हिंदी (Hindi)
    - 中文 (Chinese)
    - 日本人 (Japanese)
* User Role Separated
* Multi Domains

## Vendors

* Spring Framework (v5)
* Hibernate
* Commons Apache.org
* FasterXml.com
* jTwig
* Bootstrap 4 / Bootswatch Themes
* CKEditor5
