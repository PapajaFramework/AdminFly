* Support multilanguage

    - Українська
    - Русский
    - Беларуская мова
    - Portuguese
    - Spanish
    - German
    - French
    - Polish
    - Norwegian
    - Indonesian
    - বাংলা (Bengali)
    - हिंदी (Hindi)
    - 中文 (Chinese)
    - 日本人 (Japanese)
    - አማርኛ (Amharic)
    - ລາວ (Lao)
    - ភាសាខ្មែរ (Khmer)
    - မြန်မာဘာသာ (Burmese)
    - ภาษาไทย (Thai)
* User Role Separated
* Multi Domains

* Hikari Connection Pool
* Commons Apache.org
* FasterXml.com
* jTwig
* Bootstrap 4 / Bootswatch Themes
